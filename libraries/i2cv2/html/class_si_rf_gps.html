void SiRfGps::switchToNmea ( void  )

Switches from binary to NMEA protocol and sets message output rates and bit rate on the port.

Request the following NMEA data at 4800 bits per second: GGA ON at 1 sec, GLL OFF, GSA ON at 1 sec, GSV ON at 5 sec, RMC ON at 1 sec, VTG OFF, MSS OFF, ZDA OFF.
